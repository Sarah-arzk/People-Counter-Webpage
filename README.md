# People Counter Webpage

## Description
A vibrant webpage to track people entering. Displays a live counter, increment and save buttons, and logs previous entries. Features a full-screen animated GIF background with stylish buttons and smooth hover effects.

## Features
- Live counter display
- Increment button to increase count
- Save button to log previous entries
- Animated GIF background covering full screen
- Stylish buttons with hover effects

## Technologies Used
- HTML
- CSS
- JavaScript
